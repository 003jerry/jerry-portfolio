# Jerry - AI产品落地创作者 个人作品集

一个简约高级的个人作品集网站，展示AI产品实践与落地作品。

## 项目特点

- **响应式设计** - 完美适配PC端和移动端
- **炫彩星空背景** - 纯CSS实现的动态星空效果
- **流畅动画** - 滚动揭示、悬浮效果、打字机光标等
- **联系弹窗** - 一键复制邮箱功能

## 页面结构

1. **首页** - 个人形象展示 + 核心slogan + 导航按钮
2. **项目页** - 相亲竞争力测评网站展示
3. **关于我** - 个人介绍 + 亮点卡片
4. **联系我** - 邮箱联系 + GitHub链接

## 技术栈

- HTML5
- CSS3 (纯CSS星空动画)
- JavaScript (原生)

## 本地运行

直接在浏览器中打开 `index.html` 即可预览。

## 部署到GitHub Pages

1. 在GitHub上创建新仓库
2. 将此文件夹推送到仓库
3. 进入仓库 Settings → Pages
4. 选择 "Deploy from a branch" → main
5. 等待几分钟即可访问
